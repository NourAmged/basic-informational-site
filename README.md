# Basic Informational Site

This is a simple informational website project. It contains multiple HTML pages that provide basic information about various topics.

## Project Structure

The project has the following structure:

```
index.js
pages/
    404.html
    about.html
    contact-me.html
    index.html
```

### Files:
- **index.js**: main simple server
- **pages/**: Contains the HTML pages for the site.
  - **404.html**: 404 error page.
  - **about.html**: About Me page.
  - **contact-me.html**: Contact Me page.
  - **index.html**: The homepage of the site.
